## Dev Setup

- [install](https://www.jetify.com/docs/devbox/installing_devbox/#install-devbox) [devbox](https://www.jetify.com/devbox)
- `devbox shell`
- `npx astro dev`

## Dev Hints

- setup upstream (for baseline code merging): `git remote add upstream git@github.com:astrogon/astrogon.git`
- fetch upstream: `git fetch upstream`
- checkout `website` branch: `git checkout website`
- merge upstream `git merge upstream/main`
